Ticket API Route Information

Base URL

https://localhost:443

Access Modifiers

Public - Anyone can access this route

Private - Only logged in users can access this route

Admin - Only logged in admin users can access this route

Template Routes

  • POST /api/template - Create a new template (ADMIN)
  • GET /api/template - Get all templates (ADMIN)
  • GET /api/template/:id - Get template by it's id number (ADMIN)
  • PUT /api/template/:id - Update a template by it's id number (ADMIN)
  • DELETE /api/template/:id - Delete a template by it's id (ADMIN)

Ticket Routes

  • POST /api/ticket - Creates new ticket (ADMIN)
  • GET /api/ticket - Returns all tickets (ADMIN)
  • GET /api/ticket/:id - Get a ticket by it's id number (PRIVATE)
  • PUT /api/ticket/:id - Update a ticket by it's id number (ADMIN)
  • DELETE /api/ticket/:id - Delete a ticket by it's id number (ADMIN)
  • GET /api/ticket/status/open - Get tickets with an OPEN status (PRIVATE)
  • GET /api/ticket/status/closed - Get tickets with a CLOSED status (ADMIN)
  • GET /api/ticket/status/in-progress - Get tickets with an IN-PROGRESS status (ADMIN)

User Routes

  • POST /api/user/login - Login a user (PUBLIC)
  • POST /api/user/register - Register a user (ADMIN)
  • GET /api/user/:id - Get a user by it's id number (ADMIN)
  • PUT /api/user/:id - Update a user by it's id number (ADMIN)
  • DELETE /api/user/:id - Delete a user by its id number (ADMIN)
